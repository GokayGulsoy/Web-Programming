SYSTEM COURSES

Operating Systems

Operating systems is one of the most important system courses
in nearly all computer sicence departments it deals with the design
and implementation of process synchronization,thread level concurrency,
memory management,virtual memory and file system implementation
issues at system level.

Computer Architecture

Computer architecture is antoher systems course that deals with the
design and implementation of instruction set architectures (ISA) for
digital computers as well as the design of microarchitecture which
means the design of central processing unit of computer, and finally design
and implementation of memory systems for computers.

Computer Networks

Computer Networks is antoher important systems course in computer science
departments that deals with the data communication between hosts and clients
todays well known layered approach is known as TCP/IP protocol suite in which
network is categorized into five layers consisting of Application layer, Transport layer,
Network layer, Data-Link layer and Physical layer. Each layer deals with a different
service that is required for the communication of computers over a network.

THEORATICAL COURSES

Theory Of Computation

Theory of Computation is an theoratical subfield of computer science
in which theoratical models for todays modern digital computers are
constructed such as Deterministic finite automatas, Non deterministic finite
automatas, Push-down automatas, and Turing machines. Also it deals with
the categorization of languages such as regular languages, context-free languages,
and recursively enumerable languages.

Analysis and Design Of Algorithms

Analysis and Design of Algorithms is one of the most important subfields
of computer science that deals with the time and space complexity analysis of
different type of algorithms. It is the field in which computer scientists conducts
research to discover better algorithms in terms of space and time complexities to solve
computationaly expensive problems which are encountered in the many other fields.

Discrete Mathematics

Discrete Mathematics is another theoratical subfield of computer science
which serves a mathematical basis for abstract concepts in computer science
Some of the topics included in Discrete Mathematics are propositional logic,
quantifiers, sequences, series, summations, proof techniques, Graphs, Trees,
Reccurence relations. Fundamental aim of this field is to let computer science students
gain sufficient amount of abstraction level for designing a solution to problems
which can be encountered throughout their study.

PROGRAMMING COURSES

Object Oriented Programming

Object-Oriented Programming Programming is important in computer science
because many real-life problems can be modeled in terms of objects, some
important concepts in object orientation are inheritence,polymorphism, and
enapsulation becasue these features allow resusable software development
so that software developers don't need to write the same or similar code over
and over again.

Concepts Of Programming Languages

Programming Languages is another important subject inside Programming
courses category, because only knowing programming may not be sufficient to fully
grasp the concepts that are used by different programming language paradigms
paradigms. There are a lot of different programming languages from different
paradigms examples of these paradigms can be procedural languages, object-oriented
languages, functional languages, logic languages. Also it deals with the concepts that are
used by different programming languages such as scoping, binding, control-flow,
data abstraction, concurreny and many more.

Compilers

Compiler design and construction is said to be the corner stone of Programming courses,
compiler is a systems program that converts high level programming language into
assembly language for target machine architecture. This translation process consists of
many phases such as lexical analysis, semantic analysis, intermediate-code generation,
instruction selection, instruction scheduling, and register allocation.
